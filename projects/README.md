## How Projects Are Organized

Each project lives in its own subdirectory with a `README.md` describing:
- **Objective** – What problem the project solves
- **Dataset** – Data used (with source)
- **Approach** – Methods and algorithms applied
- **Results** – Outcomes, metrics, and insights
- **How to Run** – Instructions to reproduce the results
